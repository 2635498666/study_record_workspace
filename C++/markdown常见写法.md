# Markdown 常见写法与 Typora 快捷键

> 适用于在 Windows 版 Typora 中记录学习笔记。Markdown 标记通常使用英文半角符号。

## 一、常见 Markdown 语法

### 1. 标题

`#` 的数量表示标题级别，`#` 后需要有一个空格。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

建议一篇笔记只使用一个一级标题，正文内容从二级标题开始分层。

### 2. 粗体、斜体与删除线

```markdown
**粗体文字**
*斜体文字*
***粗斜体文字***
~~删除线文字~~
```

显示效果：**粗体文字**、*斜体文字*、***粗斜体文字***、~~删除线文字~~。

### 3. 列表

无序列表：

```markdown
- 第一项
- 第二项
  - 缩进后的子项
```

有序列表：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

列表符号后需要有一个空格。在 Typora 中连续按两次 `Enter` 可以退出列表。

### 4. 任务列表

```markdown
- [ ] 尚未完成
- [x] 已经完成
```

显示效果：

- [ ] 尚未完成
- [x] 已经完成

### 5. 引用

```markdown
> 这是一段引用。
>
> 引用中也可以包含多个段落。
```

显示效果：

> 这是一段引用。

### 6. 行内代码与代码块

行内代码使用一对反引号：

```markdown
使用 `std::vector` 保存数据。
```

显示效果：使用 `std::vector` 保存数据。

代码块使用三对反引号，并在开头注明语言以启用语法高亮：

````markdown
```cpp
#include <iostream>

int main() {
    std::cout << "Hello, Markdown!\n";
    return 0;
}
```
````

### 7. 链接与图片

```markdown
[链接文字](https://example.com)

![图片说明](图片路径或图片网址)
```

在 Typora 中也可以直接把图片拖入文档，或者粘贴剪贴板中的截图。移动 Markdown 文件前，应留意相对路径引用的图片是否仍然有效。

### 8. 表格

```markdown
| 姓名 | 成绩 | 备注 |
| --- | ---: | :---: |
| 小明 | 95 | 优秀 |
| 小红 | 88 | 良好 |
```

对齐方式：

- `---`：默认左对齐
- `---:`：右对齐
- `:---:`：居中对齐

也可以在 Typora 中按 `Ctrl + T` 快速插入表格。

### 9. 分隔线

单独一行输入三个或更多连字符：

```markdown
---
```

### 10. 换行与分段

- 段落之间留一个空行，结构最清晰。
- 在 Typora 编辑界面中按 `Enter` 开始新段落。
- 按 `Shift + Enter` 可以插入软换行，不开始新的段落。

### 11. 转义特殊符号

如果希望显示 Markdown 符号本身，可以在符号前添加反斜杠：

```markdown
\*这段文字不会变成斜体\*
\# 这也不会变成标题
```

## 二、Typora 常用快捷键（Windows）

### 文件与编辑

| 功能 | 快捷键 |
| --- | --- |
| 新建文档 | `Ctrl + N` |
| 打开文档 | `Ctrl + O` |
| 保存 | `Ctrl + S` |
| 另存为 | `Ctrl + Shift + S` |
| 关闭当前文档 | `Ctrl + W` |
| 撤销 | `Ctrl + Z` |
| 重做 | `Ctrl + Y` |
| 剪切 | `Ctrl + X` |
| 复制 | `Ctrl + C` |
| 粘贴 | `Ctrl + V` |
| 查找 | `Ctrl + F` |
| 替换 | `Ctrl + H` |
| 全选 | `Ctrl + A` |

### 排版与插入

| 功能 | 快捷键 |
| --- | --- |
| 一级至六级标题 | `Ctrl + 1` 至 `Ctrl + 6` |
| 恢复普通段落 | `Ctrl + 0` |
| 加粗 | `Ctrl + B` |
| 斜体 | `Ctrl + I` |
| 下划线 | `Ctrl + U` |
| 插入链接 | `Ctrl + K` |
| 插入图片 | `Ctrl + Shift + I` |
| 插入表格 | `Ctrl + T` |
| 插入代码块 | `Ctrl + Shift + K` |
| 插入公式块 | `Ctrl + Shift + M` |
| 插入引用 | `Ctrl + Shift + Q` |
| 增加缩进 | `Ctrl + ]` |
| 减少缩进 | `Ctrl + [` |
| 清除格式 | `Ctrl + \\` |

### 视图

| 功能 | 快捷键 |
| --- | --- |
| 切换源码模式 | `Ctrl + /` |
| 专注模式 | `F8` |
| 打字机模式 | `F9` |
| 全屏 | `F11` |

> 不同 Typora 版本或自定义配置的快捷键可能略有差异。可以打开菜单栏查看命令右侧标出的实际快捷键。

## 三、简单的学习笔记模板

````markdown
# 笔记标题

## 今日目标

- [ ] 目标一
- [ ] 目标二

## 核心知识点

用自己的话概括今天学到的内容。

## 示例代码

```cpp
// 在这里记录示例代码
```

## 遇到的问题

1. 问题是什么？
2. 根本原因是什么？
3. 最后如何解决？

## 总结与心得

记录最重要的结论，以及下次复习时需要注意的地方。
````
